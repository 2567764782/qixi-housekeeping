# 🎯 Vercel 项目导入页面操作指南

## 你现在在：Vercel New Project 页面

页面地址：`vercel.com/new`

---

## ✅ 第一步：导入 qixi-housekeeping 仓库

### 在页面左侧找到仓库列表

你会看到类似这样的列表：

```
Import Git Repository

qixi-housekeeping    [Import] ← 点击这个！
qixi                 [Import]
(其他仓库)            [Import]
```

### 操作：

**点击第一个 qixi-housekeeping 仓库右侧的 "Import" 按钮**

这个仓库应该标注了 "1h ago"（1小时前）

---

## 📋 接下来会发生什么

点击 Import 后，你会进入项目配置页面，在那里我们需要：

### 1. 设置 Root Directory（重要！）
- 展开 "Root Directory" 设置
- 选择 **"server"** 文件夹

### 2. 配置环境变量
- DATABASE_URL
- SUPABASE_URL
- SUPABASE_ANON_KEY
- JWT_SECRET
- NODE_ENV

### 3. 点击 Deploy 按钮

---

## ⚠️ 注意事项

- **一定要点击 qixi-housekeeping 的 Import**
- 不要点击其他仓库的 Import
- 第一个仓库标注了 "1h ago"，这是正确的仓库

---

## 🎯 现在就点击 Import 按钮

点击后，告诉我你看到了什么配置页面！
