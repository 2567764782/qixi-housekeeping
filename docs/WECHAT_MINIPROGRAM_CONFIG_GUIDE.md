## 3. 配置服务器域名

### 步骤 1：进入开发设置

```
┌─────────────────────────────────────────┐
│  微信公众平台                            │
├─────────────────────────────────────────┤
│  左侧菜单：                              │
│  ├─ 管理                                │
│  ├─ 内容管理                            │
│  ├─ 统计                                │
│  ├─ 「开发」 ← 点击这里                 │
│  │  └─ 「开发设置」 ← 点击这里         │
│  └─ 设置                                │
└─────────────────────────────────────────┘
```

### 步骤 2：找到服务器域名配置

```
┌─────────────────────────────────────────┐
│  开发设置                                │
├─────────────────────────────────────────┤
│                                          │
│  开发者ID                               │
│  ┌────────────────────────────────┐   │
│  │ AppID: wx1234567890abcdef      │   │
│  │            [复制]               │   │
│  │                                │   │
│  │ AppSecret: ************        │   │
│  │            [重置]               │   │
│  └────────────────────────────────┘   │
│                                          │
│  服务器域名                              │
│  ┌────────────────────────────────┐   │
│  │ ↓ scroll down 找到这里         │   │
│  └────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### 步骤 3：添加 request 域名

```
┌─────────────────────────────────────────┐
│  request 合法域名                        │
├─────────────────────────────────────────┤
│  ┌────────────────────────────────┐   │
│  │ https://your-domain.com       │   │
│  │              [删除]             │   │
│  │                                │   │
│  │ https://api.example.com        │   │
│  │              [删除]             │   │
│  └────────────────────────────────┘   │
│                                          │
│  [添加域名] ← 点击这个                   │
└─────────────────────────────────────────┘
```

点击「添加域名」后：

```
┌─────────────────────────────────────────┐
│  添加 request 域名                       │
├─────────────────────────────────────────┤
│  请填写 request 合法域名                │
│                                          │
│  域名：[https://your-domain.com]       │
│                                          │
│  [确认] ← 点击这个                       │
└─────────────────────────────────────────┘
```

**重要提示：**
- 必须使用 HTTPS
- 不要包含端口号
- 域名需要备案

### 步骤 4：添加 uploadFile 域名

同样的步骤添加 uploadFile 域名：

```
┌─────────────────────────────────────────┐
│  uploadFile 合法域名                     │
├─────────────────────────────────────────┤
│  ┌────────────────────────────────┐   │
│  │ https://your-domain.com       │   │
│  │              [删除]             │   │
│  └────────────────────────────────┘   │
│                                          │
│  [添加域名] ← 点击这个                   │
└─────────────────────────────────────────┘
```

### 步骤 5：添加 downloadFile 域名

```
┌─────────────────────────────────────────┐
│  downloadFile 合法域名                   │
├─────────────────────────────────────────┤
│  ┌────────────────────────────────┐   │
│  │ https://your-domain.com       │   │
│  │              [删除]             │   │
│  └────────────────────────────────┘   │
│                                          │
│  [添加域名] ← 点击这个                   │
└─────────────────────────────────────────┘
```

### 步骤 6：添加 socket 域名（如果需要）

```
┌─────────────────────────────────────────┐
│  socket 合法域名                         │
├─────────────────────────────────────────┤
│  ┌────────────────────────────────┐   │
│  │ wss://your-domain.com         │   │
│  │              [删除]             │   │
│  └────────────────────────────────┘   │
│                                          │
│  [添加域名] ← 点击这个                   │
└─────────────────────────────────────────┘
```

**注意：** socket 域名使用 `wss://` 而不是 `https://`

### 步骤 7：保存配置

```
┌─────────────────────────────────────────┐
│  服务器域名                              │
├─────────────────────────────────────────┤
│                                          │
│  request 合法域名：                      │
│  ✅ https://your-domain.com             │
│                                          │
│  uploadFile 合法域名：                   │
│  ✅ https://your-domain.com             │
│                                          │
│  downloadFile 合法域名：                 │
│  ✅ https://your-domain.com             │
│                                          │
│  socket 合法域名：                       │
│  ✅ wss://your-domain.com               │
│                                          │
│  [保存并提交] ← 点击这个                │
└─────────────────────────────────────────┘
```

点击「保存并提交」后，配置会立即生效。

- 每个模板需要审核
- 用户需要授权才能发送

## 🆘 常见问题

### Q1: 域名添加后提示"不合法"？

**A:** 检查：
- 是否使用 HTTPS
- 域名是否已备案
- 域名格式是否正确

### Q2: AppSecret 忘记保存了？

**A:** 只能重新生成：
- 进入开发设置
- 点击 AppSecret 旁边的「重置」
- 扫码验证
- 复制新的 AppSecret

### Q3: 支付申请被拒绝？

**A:** 检查：
- 营业执照是否清晰
- 企业信息是否正确
- 结算账户是否正确

### Q4: 订阅消息申请被拒绝？

**A:** 检查：
- 模板内容是否合规
- 使用场景描述是否清晰
- 服务类目是否正确
