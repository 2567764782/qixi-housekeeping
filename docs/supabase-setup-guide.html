🗄️ Supabase 数据库创建完整教程

步骤 1：访问 Supabase 官网

打开浏览器，访问：https://supabase.com

点击右上角的 "Start your project" 按钮

步骤 2：使用 GitHub 登录

选择 "Continue with GitHub"

如果你还没登录 GitHub，会提示你先登录

授权 Supabase 访问你的 GitHub 账号

步骤 3：创建组织（Organization）

第一次使用需要创建组织：

  • Organization Name: 填写 qixi-housekeeping
  • 点击 "Create organization"

步骤 4：创建项目

点击 "New Project" 按钮

填写以下信息：

  • Name: qixi-housekeeping
  • Database Password: 设置一个复杂密码（一定要记住！）
  • Region: 选择 Northeast Asia (Tokyo) 或 Southeast Asia (Singapore)
  • Pricing Plan: 选择 Free

点击 "Create new project"

⚠️ 重要提示：数据库密码需要包含：
  • 至少 8 个字符
  • 包含大小写字母
  • 包含数字
  • 包含特殊符号（如：!@#$%^&*）

示例密码：QixiHousekeeping2024!@#

步骤 5：等待项目初始化

项目创建需要 1-2 分钟

你会看到一个加载动画，显示 "Setting up project..."

完成后会自动跳转到项目控制台

步骤 6：获取数据库连接信息

项目创建完成后：

  1. 点击左侧菜单的 "Settings"（齿轮图标）
  2. 点击 "Database"
  3. 向下滚动找到 "Connection string" 部分
  4. 选择 "URI" 标签
  5. 点击 "Copy" 按钮复制连接字符串
# 连接字符串格式： postgresql://postgres.xxxxx:你的密码@aws-0-ap-northeast-1.pooler.supabase.com:6543/postgres
⚠️ 注意：
  • 复制后需要把 [YOUR-PASSWORD] 替换成你设置的密码
  • 密码中如果有特殊字符，需要 URL 编码

✅ 完成！

现在你已经有了：

  • ✅ Supabase 项目
  • ✅ 数据库连接字符串（DATABASE_URL）
