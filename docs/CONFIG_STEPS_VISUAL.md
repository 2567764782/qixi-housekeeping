## 常见问题

### ❌ 提示 "AppID 格式不对"

```
正确格式：wx1234567890abcdef
- wx 开头
- 共 18 个字符
- 后面是 16 位字母或数字

错误示例：
- wx123（太短）
- WX1234567890ABCDEF（大写不对）
- abc1234567890abcdef（没有 wx 前缀）
```

### ❌ 提示 "Supabase URL 格式不对"

```
正确格式：https://xyzabc.supabase.co
- https:// 开头
- .supabase.co 结尾

错误示例：
- https://supabase.com（不对，这是官网）
- xyzabc.supabase.co（缺少 https://）
- https://xyzabc.supabase.com（co 不是 com）
```
