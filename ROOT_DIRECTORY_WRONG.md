# ❌ Root Directory 选择错误

## 当前选择

你选择了 **`assets`** 文件夹 ❌

这是错误的！

---

## 为什么错误？

**`assets` 文件夹**只包含：
- 图片文件
- 静态资源

不是后端代码！

---

## 🔍 应该选择哪个？

### 正确的后端代码位置

后端代码应该在以下文件夹之一：

1. **`server/` 文件夹** ← 最可能
2. **`src/` 文件夹** ← 可能
3. **`backend/` 文件夹** ← 可能

---
