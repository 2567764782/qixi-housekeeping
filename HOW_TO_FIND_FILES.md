# 📁 如何在 Coze 平台找到项目文件

## 第一步：查看项目文件列表

在 Coze 平台界面，你应该能看到一个**文件列表**或**文件树**，通常显示：

```
📁 src                  ← 这是一个文件夹（前端代码）
📁 server               ← 这是一个文件夹（后端代码）
📁 config               ← 这是一个文件夹（配置文件）
📁 assets               ← 这是一个文件夹（静态资源）
📄 package.json         ← 这是一个文件
📄 vercel.json          ← 这是一个文件
📄 tsconfig.json        ← 这是一个文件
```

**文件夹图标**通常是 📁 黄色文件夹形状
**文件图标**通常是 📄 白色文件形状

---

## 第二步：如果你找不到文件列表

### 方法 A：导出整个项目（最简单）

1. 在 Coze 平台找到 **"导出"** 或 **"下载"** 按钮
2. 选择 **"导出整个项目"** 或 **"下载项目"**
3. 等待导出完成
4. 解压下载的文件
5. 你会看到所有需要的文件夹

### 方法 B：使用项目根目录

如果你在文件列表中，尝试：
- 点击 **项目根目录** 或 **根目录**
- 或点击 **项目名称**
- 查看是否有下拉菜单或展开按钮

---

## 第三步：确认文件存在

在文件列表中，你应该能看到：

### ✅ 必须有的文件夹

1. **src/** 文件夹
   - 包含：app.tsx, pages/, components/ 等

2. **server/** 文件夹
   - 包含：src/, database/, deploy.sh 等

3. **config/** 文件夹
   - 包含：index.ts, dev.ts, prod.ts

4. **assets/** 文件夹
   - 包含：image.png 等图片

### ✅ 必须有的文件

- package.json
- vercel.json
- tsconfig.json

## 💡 最简单的方法

如果不确定，直接：
1. 在 Coze 平台找 **"导出项目"** 按钮
2. 导出整个项目到本地
3. 解压后会看到所有文件
4. 然后把这些文件上传到 GitHub

这样最简单，不会遗漏任何文件！
