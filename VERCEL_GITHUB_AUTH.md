# 📋 Vercel GitHub 授权页面操作指南

## 你现在在：Vercel GitHub 应用安装页面

这个页面让你选择 Vercel 可以访问哪些 GitHub 仓库。

---

## 选项说明

### 选项 1：All repositories（所有仓库）
- ✅ 已默认选中
- Vercel 可以访问你所有的 GitHub 仓库
- 包括未来的新仓库
- 更简单，但权限更大

### 选项 2：Only select repositories（仅选择的仓库）← 推荐
- 更安全
- 只授权 Vercel 访问你指定的仓库
- 建议选择这个

---

## 🎯 推荐操作步骤

### 第一步：选择 "Only select repositories"

**点击第二个选项**：
```
○ Only select repositories
```

### 第二步：选择 qixi-housekeeping 仓库

选择后，会出现一个下拉列表或搜索框：

**在列表中找到并选择**：
```
qixi-housekeeping
或
2567764782/qixi-housekeeping
```

### 第三步：点击 Install 按钮

选择好仓库后，点击页面底部的绿色按钮：
```
Install
```

---

## 💡 快速方法

如果你觉得选择仓库麻烦，也可以：

1. **保持当前选择**："All repositories"
2. **直接点击**："Install"

这样 Vercel 会访问你所有的仓库，更简单但权限更大。

---

## 🚀 点击 Install 后

你会被重定向到 Vercel 网站，开始配置项目部署。
